# ArxRate : Veille Scientifique Automatisée

ArxRate est un système d'intelligence artificielle conçu pour automatiser l'analyse et la synthèse de publications scientifiques. À partir de mots-clés ou de thématiques, il récupère les derniers papiers sur ArXiv liés et utilise l'IA générative pour produire des résumés critiques et identifier les innovations majeures.
